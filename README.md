
 Overview
Dynamic Labyrinths is a complete multiplayer educational game designed to teach and visualize search algorithms within a dynamic environment. The project showcases complex game logic, full system architecture from the ground up, and the integration of artificial intelligence.

 Tech Stack
* Game Engine: Unity (C#)
* AI & Machine Learning: Python
* Core Concepts:Reinforcement Learning, Multiplayer Networking, Physics, Pathfinding Algorithms

## Key Features
* Multiplayer Environment: Supports multiple users interacting within the same dynamic game state.
* Autonomous AI Bots: Features custom bots that navigate the environment, trained independently using Python-based Reinforcement Learning.
* Algorithm Visualization: Provides real-time visual representation of search algorithms to enhance the educational value of the game.
* Robust Architecture: Built from scratch to seamlessly handle complex game logic, physics calculations, and client-server synchronization.
